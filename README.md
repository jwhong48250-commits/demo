🎮 StarBlast 2D — 전투기가 짱구를 물리치는 웹 슈팅 게임 만들기 (Phaser.js)

이번 글에서는 Phaser.js를 사용해 만든 간단한 2D 슈팅 게임인 StarBlast 2D를 소개합니다.
아군 캐릭터는 전투기, 적군은 짱구 이미지가 등장하며,
사방에서 몰려오는 적들을 피하고 격추하면서 점수를 올리는 게임입니다.

브라우저만 있으면 어디서든 실행할 수 있는 웹게임이라 만들기도, 플레이하기도 쉽습니다.

🚀 게임 특징
✔ 1. 전투기 조종 가능

방향키 ← ↑ ↓ → 로 이동

전투기는 이동 방향에 따라 자연스럽게 회전

SPACEBAR(스페이스 키)로 총알 발사

✔ 2. 짱구 적군 등장

짱구 얼굴 이미지가 ‘적군’ 역할

화면 4방향(위/아래/왼쪽/오른쪽)에서 무작위로 스폰

전투기 위치를 추적하며 이동

시간이 지날수록 더 빨리 나오고 HP도 증가

✔ 3. 충돌 처리

총알이 짱구를 맞추면 HP 감소 후 파괴

짱구가 전투기와 충돌하면 전투기 HP 감소

HP가 0이 되면 게임 오버

✔ 4. 점수 시스템

짱구 하나 제거할 때마다 +10점

화면 좌측 상단에 현재 점수 표시

전투기 HP도 실시간 표시됨

🕹 게임 플레이 화면 구성

중앙: 전투기 등장

사방: 짱구가 랜덤하게 생성되어 침투

좌측 상단: Score / HP 상태 표시

중앙: 게임 오버 시 빨간색 "GAME OVER" 표시

게임 자체가 매우 직관적이고 빠른テンポ의 슈팅 게임이라
짧고 재밌게 플레이하기 좋습니다.
